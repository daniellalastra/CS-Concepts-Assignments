# CS Concepts - Assignments

A collection of coding assignments from a Computer Science Concepts course, 
covering data structures, algorithms, and language concepts across Java and C.

## Projects

### Assignment 1 - Student BST (Java)
A Binary Search Tree implementation that manages student data. Reads from 
input files and supports insertion, traversal, and search operations.

### Assignment 2 - BST in C
A Binary Search Tree implementation in C with file I/O. Reads student records 
from input files and demonstrates memory management and pointer manipulation.

### Assignment 7 / Common 4 - C Data Structures
Additional C programming assignments covering data structures and file handling.

### Project - CS4308 Calculator (Java)
A calculator project built for CS4308, demonstrating parsing and language 
concepts.
